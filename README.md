# AuthenticationPages

This is a simple **Authentication Forms Project** using React, Formik, and Yup.  
It includes Login and Signup forms with validation, error messages, and redirects.

---

## Features

- **Signup Form**: First Name, Second Name, Email, Password.
- **Login Form**: Email and Password.
- **Validation**: Using Yup to check inputs.
- **Error Messages**: Show errors below each field.
- **Redirects**:
  - After Signup → go to Login page.
  - After Login → go to Home page.
- **Home Page**: Shows welcome message with user's name.
- **Logout**: Button to return to Login page.
